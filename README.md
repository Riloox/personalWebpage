# Personal Portfolio

A React + Vite starter tailored for showcasing software development skills, experience, and highlighted work.

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - start Vite dev server with fast HMR
- `npm run build` - type-check and create a production build
- `npm run preview` - preview the production build locally

## Customize

Edit the structured content in `src/data/profile.ts` to update the hero copy, skills, experience timeline, projects, and contact links. Styles live in `src/index.css` and can be adjusted or replaced with your preferred design system.
